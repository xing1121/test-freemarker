常用内建函数

1.字符串常用内建函数:

    <#assign a='Hello'/> <#assign b='World'/>
  • a:${a}
  • b:${b}
  • +:${a + b}
  • substring:${(a + b)?substring(0, 5)}
  • length:${(a + b)?length}
  • upper_case:${(a + b)?upper_case}
  • lower_case:${(a + b)?lower_case}
  • index_of:${(a + b)?index_of('W')}
  • last_index_of:${(a + b)?last_index_of('o')}
  • replace:${(a + b)?replace('W', 'WWWWWW')}
  • <#list "a|b|c|d"?split("|") as x>${x},
    • <#assign d1 = "2018-08-27 10:11:55"/> <#assign d2 = "2018-08-27"/> <#assign d3 = "10:11:55"/>
  • ${d1?datetime("yyyy-MM-dd HH:mm:ss")}
  • ${d2?date("yyyy-MM-dd")}
  • ${d3?time("HH:mm:ss")}

2.数字常用内建函数:

    <#assign num1 = 3.1415926/>
  • ${num1?string("0.##")}
  • ${num1?round}

3.list常用内建函数:

    <#assign l1 = [1,2,3,4,11,12,13,14,21,22,23,24,31]/>
  • ${l1?chunk(4)?size}
  • <#list l1?chunk(4)?first as x>${x},
    • <#assign myList=[21,13,14,15,11,81,19,81,17]/>
  • 自带正序： <#list myList?sort as x> ${'index' + x_index}:${x}
  • 自带倒序： <#list myList?sort?reverse as x> ${'index' + x_index}:${x}
  • 索引取值： <#list myList as x> ${'index' + x_index}:${myList[x_index]}

4.其他内建函数:

  • ${"123"?is_number}
  • ${123?is_number}
  • ${"123"?is_string}
  • ${haha!"哈哈"}
  • ${haha?has_content?string("yes", "no")}
  • ${("1" + "2")?is_number}
  • ${("1" + "2")?eval?is_number}
